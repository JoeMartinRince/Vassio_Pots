# Vassio — Modern Home Decor & Furniture

Vassio is a premium, modern e-commerce web application featuring handcrafted ceramic pots, minimalist fiber-glass planters, and curated indoor plants. The design is built to bring warmth, craft, and organic calm into home sanctuaries.

## 🎨 Design System & Aesthetics

* **Dominant Tones**: Warm, rich organic browns (`oklch` values tailored for a premium cozy look) that form the core backdrops, cards, and text layout styles.
* **Accent & Undertones**: Soft wasabi green (theme's `primary` color) used selectively for hover transitions, button borders, and decorative badge highlights to keep the look clean, premium, and alive.
* **Typography**: Elegant combinations of *Cormorant Garamond* (serif headings) and *Inter* (clean sans-serif body text).

## 🚀 Tech Stack

* **Framework**: [TanStack Start](https://tanstack.com/router/v1/docs/start/overview) (React Server Components + Type-safe Routing)
* **Language**: TypeScript
* **Styling**: Tailwind CSS v4 & Lucide Icons
* **Build System**: Vite

## ✨ Key Features

1. **Custom Hero Section**: Fully responsive, high-impact background slide optimized to occupy the exact remaining screen height on both mobile (`100dvh`) and desktop viewports.
2. **Interactive Elements**: Hover overlays, smooth scroll, carousel slider for products, and sliding mobile side-drawer navigation.
3. **Watch and Buy**: A modern video reel carousel showcasing plants in action with direct shopping card overlays.
4. **Interactive Shop Pages**: Responsive grid layouts, filter drawers (by categories, price range, stock levels), and type-safe routing.
5. **Clean Code & SEO**: Preconfigured with semantic HTML tags, complete meta titles/descriptions, and fast asset loading.

## 🛠️ Local Development Setup

To run the project locally on your machine:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start the Development Server**:
   ```bash
   npm run dev
   ```
   This will spin up a local development server at [http://localhost:8080/](http://localhost:8080/).

3. **Build for Production**:
   ```bash
   npm run build
   ```

4. **Preview the Production Build**:
   ```bash
   npm run preview
   ```
